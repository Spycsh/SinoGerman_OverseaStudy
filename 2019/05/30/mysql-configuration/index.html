Mysql configuration

Posted on 2019-05-30 | Comments:

Solve the problems of starting the server of Mysql, changing the initial password and connection in workbench. If you get any problems, please give feedback to us.

切换到mysql server文件夹下,开启服务

打开服务后,在mysqlworkbench中尝试连接,密码要从C:\Program Files\MySQL\MySQL Server 8.0\data下的.err文件中查看

找到你文件中类似如下log信息

root@localhost后的一串乱码就是你第一次登陆的密码。

在workbench登陆之前需要改动密码
